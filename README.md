# 🍽️ Khuda Lagche

Khuda Lagche is a modern restaurant landing page built as a practical front-end project.  
This project focuses on clean UI design, structured layout, and Tailwind CSS implementation.

🔗 Live Demo: [View](https://dsdipu.github.io/Khuda-lagche)
🔗 Repo Link: [View](https://github.com/dsdipu/Khuda-lagche)
---
## 📌 Overview

Khuda Lagche is a static restaurant landing page designed to showcase a food brand with a visually appealing interface.  
This project was developed for practical learning purposes to strengthen front-end development skills, particularly layout structuring and utility-first CSS styling using Tailwind CSS.
The website currently focuses on UI design only and does not include backend functionality or JavaScript-based interactivity.
---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS (CLI)
- Font Awesome
---

## ✨ Features

- Sticky Header
- Hero Section
- Menu Section
- Footer Section
- Hover Effects
- Clean and Structured Layout
- Utility-first CSS Styling
---

## 📂 Project Structure

```
KHUDA-LAGCHE/
│
├── Images/

│   ├── input.css

│
└── index.html
```

---

## ⚙️ Installation & Setup

To run the project locally with Tailwind CSS CLI:

```bash
npx @tailwindcss/cli -i src\input.css -o src\output.css --watch
```

Then open `index.html` in your browser.

- Some layout inconsistencies may appear on different screen resolutions.
